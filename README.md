# css city


![css-project](http://pdanxug8c.bkt.clouddn.com/css%20project.gif)

## 简介
用css画一个城市插画

- 颜色变量、渐变、字体
- 建筑物形状和排版
- 夜晚版本
- 使用sass min加入月亮和星星
- 白天和晚上功能切换
